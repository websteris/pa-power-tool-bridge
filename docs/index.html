Power Automate Power Tool

A professional JSON editor for Microsoft Power Automate flows — live save, flow tree navigation, smart rename, and a CLI bridge for scripting and AI integration.

Features

Monaco Editor

The same editor powering VS Code. Full JSON editing with syntax highlighting, bracket matching, and inline schema validation.

Live Save

Changes are patched directly to the Power Automate API — no page reloads, no copy-paste workflows.

Flow Tree

Visual tree of all actions in your flow. Click any node to jump to it in the editor instantly.

Smart Rename

Press F2 on any action key to rename it everywhere — updates runAfter dependencies and expressions automatically.

CLI Bridge

Automate your flow editing

The CLI Bridge lets scripts, Claude, and other tools read the current flow definition, watch for errors, and send commands directly to the extension — all via simple local files. No API keys, no authentication.

Install the CLI Bridge →
Read the current flow (PowerShell)
Get-Content "$env:TEMP\pa-power-tool\current-flow.json"
Trigger a save (PowerShell)
'{"command":"save"}' | Set-Content "$env:TEMP\pa-power-tool\commands.json"
Read the current flow (bash)
cat "$TMPDIR/pa-power-tool/current-flow.json"
